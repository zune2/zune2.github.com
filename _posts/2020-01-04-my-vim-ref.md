주로 쓰는 vim 명령어 정리

## basic command

| CMD	                         | DESC                      |
|--------------------------------|---------------------------|
| :w filename                    | 파일 이름으로 저장        |
| Ctrl-v 커서 이동 + y           | visual block만들고 copy   |
| p                              | paste 하기                |

## plugin이용 command

| CMD	                         | DESC                      |
|--------------------------------|---------------------------|
| Ctrl-p                         | ctrlp로 파일 찾기         |



## C/C++용 키맵
| CMD	                           | DESC                        |
|--------------------------------|-----------------------------|
| Ctrl-]                         | :cs find 3 keyword,  심벌로 이동         |
| Ctrl-t                         | 원래 위치로 이동하기                     |
| :make                          | makecmd를 실행한다.                      |

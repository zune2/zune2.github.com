bit shift 예제

왼쪽으로 시프트 : 2를 곱하는 효과 

>>> 1 << 0
1
>>> 1 << 1
2
>>> 1 << 2
4
>>> 1 << 3
8

오른쪽으로 shift : 2로 나누는 효과 

>>> 10 >> 1
5
>>> 5 >> 1
2
>>> 2 >> 1
1
>>> 1 >> 1
0

알파벳 A,B,C,D를 bit로 mapping하기

1 << (ch - 'A')

작은 bit부서 순서대로 set된다. 

A : 1 << ( 'A' - 'A')  =  1 << 0  =  1  =  0001
B : 1 << ( 'B' - 'A')  =  1 << 1  =  2  =  0010
C : 1 << ( 'C' - 'A')  =  1 << 2  =  4  =  0100
D : 1 << ( 'D' - 'A')  =  1 << 3  =  8  =  1000

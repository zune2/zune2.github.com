asciidoctor을 이용한 page 생성 기능 추가함 {uri-asciidoctor}[Asciidoctor].

== asciidoc plugin 설치, 지정

----
sudo gem install jekyll-asciidoc
----

._config.yml을 변경함
----
gems:
    - jekyll-asciidoc
----
* 3.0.1버전이라서 gems라고 적어줌.

== 소스코드 출력
[source,c]
----
#include <stdio.h>
int main(int argc, char* argv[]) {
	printf("Hello World\n");	
	return 0;
}
----
